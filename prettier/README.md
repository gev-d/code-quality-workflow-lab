**Prettier** is **a highly popular, open-source, and "opinionated" code formatter used by developers globally**. It automatically parses your code and completely reprints it from scratch according to its own strict design guidelines, taking maximum line lengths into account and wrapping lines when necessary.

For example, you write:

```
constuser={name:"Gevorg",age:25}
```

Prettier formats it as:

```
constuser= {
  name:"Gevorg",
  age:25,
};
```

It can automatically fix things such as:

- spaces and indentation
- line breaks
- quotation marks
- semicolons
- trailing commas
- maximum line length

Prettier does **not usually check whether your code logic is correct**. It mainly makes the code clean and consistently formatted.
